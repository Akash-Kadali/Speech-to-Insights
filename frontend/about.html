About This Project

A modern system that transforms speech into structured insights using NLP.

Project Overview

Speech to Insights is an end-to-end system built for the MSML 650 — Machine Learning Systems course. It converts meeting audio and transcripts into searchable, analyzable insights.

Users can upload recordings, perform semantic search, view transcripts, and explore analytics dashboards based on topic trends and participation.

System Architecture

  • Frontend: Static HTML/CSS/JS with reusable components and a small utility layer.
  • Processing: Demonstration NLP pipeline for transcription, embedding, and summarization.
  • Search: Embedding-based semantic search (demo implementation).
  • Visualization: Topic frequency, speaker activity, and word-clouds.
  • Storage: Local/session storage for demo; replace with server-side storage for production.

Motivation

Teams spend hours re-watching meetings. This project aims to:

  • Reduce manual note-taking.
  • Enable quick retrieval of decisions and action items.
  • Surface conversation trends and speaker participation.
  • Demonstrate practical ML system design on the frontend.

Key Features

  • Audio/Text Upload: Process meetings from multiple formats.
  • Semantic Search: Ask natural questions and return relevant snippets.
  • Session Explorer: Transcripts, summaries, and conversation flow.
  • Analytics: Topic and speaker metrics.
  • Profile & Theme: Simple personalization and light/dark themes.
  • Admin Tools: Data seeding and queue monitoring (demo).
