About This Project

A modern system that transforms speech into structured insights using NLP.

Project Overview

Speech to Insights is an end-to-end system built for the MSML 650 – Machine Learning Systems course. It converts meeting audio or transcripts into searchable, analyzable insights.

Users can upload recordings, perform semantic search, view session transcripts, and explore analytics dashboards based on topic trends and participation.

System Architecture

  • Frontend: Pure HTML/CSS/JS (no frameworks) with reusable UI components.
  • Processing Layer: Simulated NLP pipeline for demo purposes.
  • Search Engine: Embedding-based semantic matching (mocked for project).
  • Visualization: Topic frequency, speaker activity, word clouds.
  • Storage: Local storage & simulated session data for grading.

Motivation

Modern teams rely on virtual meetings, but reviewing long discussions is inefficient. This project aims to:

  • Reduce manual note-taking.
  • Enable fast retrieval of decisions and action items.
  • Support analytics on conversation trends.
  • Demonstrate real-world ML system design concepts.

Key Features

  • Audio/Text Upload: Process meetings from multiple formats.
  • Semantic Search: Ask natural questions and find relevant conversation snippets.
  • Session Explorer: View transcripts, summaries, and conversation flow.
  • Analytics Dashboard: Topic frequency, speaker participation, and keywords.
  • Profile & Theme: User personalization and light/dark theme.
  • Admin Tools: Simulated data seeding and queue monitoring.
