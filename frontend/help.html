Frequently Asked Questions

1. What kind of files can I upload?

You can upload .mp3, .wav, .m4a, .txt, or .pdf files in the Upload page.

2. How does semantic search work?

The search engine uses embedding similarity to match your natural-language query against all transcripts. It retrieves the most relevant segments from your meetings.

3. What are “sessions”?

Sessions represent individual meetings or recordings you uploaded.
You can view transcripts, highlights, and insights inside each session.

4. Why am I seeing demo data?

This project includes optional demo mode to help instructors and graders quickly explore features without uploading their own files.

5. Where is my data stored?

For this MSML 650 project version, data is stored locally or simulated.
No real cloud storage or user authentication is used.

How to Use This System

Upload

Go to the Upload page to add a new meeting. Fill in the fields and select your file.

Search

Use the Search page to ask natural questions like:
“What decisions were made about pricing?”

Sessions

View all uploaded meetings, open any session, and explore the transcript and highlights.

Analytics

Visual dashboards show topic frequency, speaker participation, and keyword patterns.

Admin Panel

Used to simulate queue status and demo data for presentation purposes.
