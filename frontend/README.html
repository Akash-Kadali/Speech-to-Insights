üìå Introduction

Speech to Insights is a frontend-driven demonstration of a real-world ML system that transforms meeting audio or transcripts into structured insights.

The system allows users to upload audio/text files, perform semantic search across transcripts, browse past sessions, and visualize analytics such as topic frequency, speaker participation, and keyword trends.

This project focuses on clean system design, user experience, and mock ML pipeline simulation suitable for an academic environment.

üéØ Project Goals

  • Create an intuitive UI for uploading and processing meeting data
  • Provide semantic search functionality using natural language queries
  • Present organized transcripts with extracted highlights
  • Visualize conversation data with charts and word clouds
  • Demonstrate ML system design concepts in a simplified frontend

üèó System Architecture (Frontend Simulation)

This project includes:

  • Upload Module ‚Äì Accepts audio & text, simulates processing
  • Semantic Search Engine ‚Äì Embedding-style similarity (mocked)
  • Session Explorer ‚Äì Views transcripts & highlights
  • Analytics Dashboard ‚Äì Topic frequency, participation, word cloud
  • Profile Page ‚Äì User preferences stored locally
  • Admin Panel ‚Äì Queue status + demo data controls

The system is designed to follow the high-level structure of ML production systems taught in MSML 650.
